# CAPM - Certified Associate in Project Management

**Offered by:** Project Management Institute

**Duration:** 23 hours of preparation. 3 hours examination.

**Exam details:** Online. 150 questions. - [Exam Content PDF](https://arclib.netlify.app/References/PMI/CAPM-Exam%20Content%20Outline-english.pdf)

**Fee structure:**

| Fee description                     | Fee amount     |
| ----------------------------------- | -------------- |
| 23 hours exam prepation (mandatory) | 24,360 INR     |
| Practice exam (optional)            | 6,003 INR      |
| Exam fee (certification exam)       | 18,270 INR     |
| **Total fee amount**                | **48,633 INR** |
